# Your AI Teacher 👩‍🏫

An AI-powered educational assistant that:
- 📘 Generates notes from PDFs or topics
- 🧠 Creates flashcards & quizzes
- 📈 Tracks learning progress with a Kanban-style planner

## 📦 Tech Stack
- HTML, CSS, JS (frontend)
- Node.js + Express (backend)
- MongoDB (storage)
- Hugging Face API (AI)
